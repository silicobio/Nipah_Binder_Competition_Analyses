# Post-Hoc Analyses of the Adaptyv Bio Nipah Virus Competition Submissions

<h3 align="right">Silico Biosciences</h3>


This repository contains code and results for post-hoc analyses of the Adaptyv Bio Nipah Virus Binder Competition submissions. The analyses include binding affinity scoring of submitted protein candidates using the HADDOCK3 scoring on the provided Boltz-2 complex structures.


- Base Dataset: https://proteinbase.com/collections/nipah-binder-competition-all-submissions

- Processed Dataset: [data/proteinbase_collection_nipah-binder-competition-all-submissions_processed.csv](data/proteinbase_collection_nipah-binder-competition-all-submissions_processed.csv)

- Binding Affinity Data: [data/haddock3_scores.csv](data/haddock3_scores.csv)
